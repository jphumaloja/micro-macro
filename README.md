# Micro-Macro Backstepping Control of Large-Scale Hyperbolic Systems

The purpose of these MATLAB codes is to reproduce the simulation results from Section VI of the paper "Micro-macro backstepping control of large-scale hyperbolic systems" by Jukka-Pekka Humaloja and Nikolaos Bekiaris-Liberis. A preprint is available in [arXiv](https://arxiv.org/abs/2510.12456).

The mathematical background and the documentation for the codes are desribed in the file "codesdoc.pdf.".

## Requirements

The codes require MATLAB with Symbolic Math Toolbox.

## Usage

The simulation results of Sections VI.A and VI.B can be reproduced by running the scripts ExA and ExB.m, respectively.
